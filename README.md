# 📋 剪贴板历史悬浮窗

一款 Windows 桌面工具，实时记录剪贴板文本历史，以悬浮窗形式始终置顶展示。点击条目即可将文本直接输入到当前光标位置。使用模拟键盘输入的方式注入文本，**能绕开部分教育网站、考试系统等对粘贴（Ctrl+V）的限制**。支持智能透明度、快捷键显隐、自动去重。

## ✨ 功能

- **实时监控**：自动记录每次复制（Ctrl+C）的文本，最多保留 30 条
- **点击即输入**：点击历史条目，通过模拟键盘输入（`SendInput`）将文本直接键入当前输入框，不经过剪贴板，**可绕开禁止粘贴复制的限制**
- **自动去重**：重复内容自动移至顶部，不产生多余条目
- **智能透明度**：鼠标未悬停时 20% 透明度，悬停后 100% 不透明；可锁定为始终不透明
- **全局快捷键**：默认 `Alt+P` 切换显隐，快捷键可自定义
- **系统托盘**：最小化到托盘驻留，右键菜单快捷操作
- **无边框悬浮**：始终置顶，可拖拽移动

## 🚀 快速开始

### 方式一：直接运行（推荐）

下载 `剪贴板历史悬浮窗.exe`，双击运行即可。首次运行自动生成 `config.json` 配置文件。

### 方式二：从源码运行

```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 运行
python main.py
```

**环境要求**：Python 3.7+，Windows 10/11

### 打包

```bash
pyinstaller --onefile --windowed --name "剪贴板历史悬浮窗" main.py
```

产物在 `dist/` 目录下。

## 🎮 使用说明

| 操作 | 方式 |
|------|------|
| 显示/隐藏窗口 | 按 `Alt+P`（可自定义） |
| 切换显隐 | 双击托盘图标 |
| 粘贴文本 | 点击列表中的条目 |
| 删除单条 | 点击条目右侧的 `×` |
| 全部清空 | 点击底部「全部清空」按钮或托盘菜单 |
| 移动窗口 | 拖拽标题栏或列表空白区域 |
| 锁定透明度 | 鼠标悬停标题栏 → 点击 📌，变为 🔒 后始终不透明 |
| 隐藏到托盘 | 鼠标悬停标题栏 → 点击 ➖ |
| 修改快捷键 | 托盘右键 → 设置快捷键 |
| 退出程序 | 托盘右键 → 退出 |

## ⌨️ 快捷键设置

1. 右键系统托盘图标
2. 点击「设置快捷键...」
3. 在输入框中按下你想要的组合键（如 `Ctrl+Shift+V`）
4. 点击确定

快捷键保存在同目录的 `config.json` 中。

## 🏗️ 项目结构

```
clipboard_history/
├── main.py               # 程序入口
├── history_manager.py    # 历史记录管理（存储、去重、裁剪）
├── clipboard_monitor.py  # 剪贴板监控（定时轮询）
├── floating_window.py    # 悬浮窗 UI
├── system_tray.py        # 系统托盘
├── hotkey_manager.py     # 全局热键（Win32 API）
├── settings_dialog.py    # 快捷键设置对话框
├── requirements.txt      # Python 依赖
└── config.json           # 用户配置（自动生成）
```
